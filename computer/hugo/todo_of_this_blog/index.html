Wanted Features

hugo hierarchy categories display

The categories of hugo is only one-level (It’s called Taxonomies). But I want the categories like the following image. It’s very common feature in another static site generator hexo image src: blog of morris821028

I had survey some feasible method to fitting my requirement, but I’m still solving this problem.

Method 1

Use a prefix string as main category - Implement with simple concatenated category string

Example:

  • blog (Main Category)
  • blog/hugo (Sub Category)

  <!-- Taxonomies (default is tag and categories) -->
  {{ range $key, $value := .Site.Taxonomies }} {{ $len := len $value }} {{ if (not ( eq $len 0 ) ) }}
  <div class="section taxonomies">
    <header>
      <div class="title"><b>{{ $key | humanize }}</b></div>
    </header>

    <div class="content">
      <ul>
        {{ range $value.Alphabetical }}
        <!--  range first 15 $value.ByCount -->
        <li><a href="{{$.LanguagePrefix }}/{{ $key }}/{{ .Name | urlize }}">{{ .Name }} ({{.Count}})</a></li>
        {{ end }}
      </ul>
    </div>
  </div>
  {{ end }} 
  {{ end }}
