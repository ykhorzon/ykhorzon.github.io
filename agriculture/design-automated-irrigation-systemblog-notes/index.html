Concepts before start

Before we start to do the works (no matter design or implement), we need to set up our mind set and objectives. It can prevent us from wasting effort.

MVP, Minimum Viable Product

minimum viable product (MVP) is a product with just enough features to gather validated learning about the product and its continued development.

  • For startups, how to know the enough features is a art.
  • For engineer / design, how to implemtent those enough features elegantly and efficiently
