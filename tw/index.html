Matplotlib 基本介紹

25 Oct, 2017 python matplotlib drawing

Introduction 以下圖片為 pyplot 的重要元件，其中最重要的為 Figure 、 Axes 、 Axis 與 Artist Figure Figure 包含 Axes 和 canvas。Axes 為會被畫出來的角色，如 Title 、Figure legends … etc fig = plt.figure() # an empty figure with no axes fig.suptitle('No axes on this figure') # Add a title so we know which it is fig, ax_lst = plt.subplots(2, 2) # a figure with a 2x2 grid of Axes 可以畫出一下的圖片 Axes Axes 是 figure 上面一個區塊的呈現資料的地方，就是你畫出來的»東西»。其中每個 Axes 會包含至少2個以上的 Axis，並且可以設置資料呈現的limits, set_xlim(), set_ylim()，此外圖片標題 title 與 x-label、y-label也是他負責。
